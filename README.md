# ⚽ Subasta Futbolera Simulator (Open Source)
Inspirado en la dinámica del programa argentino **412**.

## 🎯 Objetivo
Un motor de simulación de partidos de fútbol basado en estadísticas reales de jugadores subastados.

## 🛠️ Roadmap
- [ ] Integración con APIs de datos de fútbol.
- [ ] Motor de subastas en tiempo real.
- [ ] Algoritmo de simulación (Medias vs. Atributos específicos).
- [ ] Sistema de "Química" de equipo.
